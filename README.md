# WeChat MiniProgram Project 02

## 项目概述
微信小程序 - 工具类应用，用于承接微信搜索流量并通过广告变现。

## 当前状态
🚀 **开发中** - 个税计算器 2026

## 项目定位
**个税计算器 2026** - 为上班族提供个税、社保、公积金精准计算工具

## 核心功能
- 个人所得税计算（2026最新税率）
- 社保公积金计算
- 年终奖计算优化
- 税后工资反推
- 个税专项扣除计算
